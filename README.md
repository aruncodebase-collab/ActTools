# ActTools
ActuarialTools for the modern general insurance actuary

## Features
- **Core Actuarial Math**: Discounting, present values, accumulated value.
- **IFRS 17 Tools**: Risk Adjustment (RA) calculations using Confidence Level (VaR), Tail VaR (TVaR), and Cost of Capital.
- **Demographics**: Life tables and mortality modeling (Planned).
